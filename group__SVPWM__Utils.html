The switching times T1 and T2 determine how long we apply the two active space vectors in the current sector, and T0 is the remaining zero vector time. For center-aligned PWM, T0 is evenly split at the start and end of each PWM cycle.

Each sector corresponds to a pair of adjacent active vectors:

  • Sector 0 [100]: V1 (0°), V2 (60°)
  • Sector 1 [110]: V2 (60°), V3 (120°)
  • Sector 2 [010]: V3 (120°), V4 (180°)
  • Sector 3 [011]: V4 (180°), V5 (240°)
  • Sector 4 [001]: V5 (240°), V6 (300°)
  • Sector 5 [101]: V6 (300°), V1 (360° or 0° again)

The duty cycles (Ta, Tb, Tc) for phases A, B, and C are assigned based on which sector we're in. This mapping ensures proper synthesis of the reference vector using the two active vectors.
